# robosys2022
 * ロボットシステム学、練習ディレクトリ

![test](https://github.com/kamemattari/robosys_plus/actions/workflows/test.yml/badge.svg)

* インストール方法
```
$ git clone https://github.com/kamemattari/robosys2022.git
$ cd robosys2022
```

# plusコマンド

標準入力から読み込んだ数字を足していく。

 * 使い方
```
$ seq 3| ./plus
6 #1から3までの和#
```

# 必要なソフトウェア
 * Python ver [3.7, 3.8,3.9,3.10]

# テスト環境
